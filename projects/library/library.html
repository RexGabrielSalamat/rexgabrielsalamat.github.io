Library Catalogue Management System

A custom Java desktop application focused on Object-Oriented Design and memory-efficient data serialization.

Video Demonstration

Gallery

Technical Summary

Built using Java Swing and OOP principles, this application manages diverse media types through inheritance. The primary technical focus was creating a memory-efficient XML parser using StringBuilder, avoiding heavy external libraries to ensure high performance on local systems.

Key Features

The system implements full CRUD functionality with a custom search and filtering engine. I designed a hierarchical classification system for Books, CDs, and DVDs, ensuring automated data normalization and reliable state persistence upon closing the application.

Note: This project was developed for academic purposes at the University of Stirling. Source code is withheld to uphold Academic Integrity policies.
