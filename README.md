# RG's Portfolio - Repository
## Repository Structure

| File / Folder | Description 
| :--- | :--- |
| **`assets/`** | Central storage for all media, including project demo videos, images, and documentations. 
| **`index.html`** | The main entry point of the website, featuring the "About Me," "Projects,", "Experience", and "Contact" sections. 
| **`certificate.html`** | A dedicated archive for professional credentials from IBM, Red Hat, LinkedIn Learning, etc. 
| **`library.html`** | Project page for the **Library Catalogue Management System**, focusing on Java-based performance and efficient data parsing. 
| **`meteor.html`** | Project page for the **Meteor Impact Simulator**, highlighting the NASA Project's API integration and physics-based modeling.
| **`cert-script.js`** | Dedicated JS logic for the certificate modal system and dynamic grid rendering. 
| **`script.js`** | Core JS logic for handling navigation and global interactive elements. 
| **`style.css`** | Centralized Tailwind CSS and custom styling for consistent look and feel across all pages. 

## Tech Stack

* **Frontend:** HTML5 and Vanilla JavaScript.
* **Styling:** Tailwind CSS for a modern, utility-first design.
* **Optimization:** Modularized code by separating CSS and JS from HTML for efficient maintainability and performance.

## Highlighted Projects

* **Meteor Impact Simulator:** Assesses planetary risk using NASA data models to estimate destruction and casualties.
* **Library Analytics System:** High-efficiency Java application prioritizing performance through custom data structures and `StringBuilder` optimization for XML/JSON parsing.
